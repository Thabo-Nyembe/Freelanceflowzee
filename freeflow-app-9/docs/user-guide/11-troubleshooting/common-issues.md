# Troubleshooting Guide

Solutions to common issues you might encounter while using KAZI.

## Login & Authentication

### Can't Log In
**Symptoms**: Login fails, password not accepted

**Solutions**:
1. Check email/password for typos
2. Click "Forgot Password" to reset
3. Check if Caps Lock is on
4. Try a different browser
5. Clear browser cache and cookies

### Two-Factor Authentication Issues
**Symptoms**: 2FA code not working

**Solutions**:
1. Ensure your device time is synced
2. Wait for new code (codes expire)
3. Use backup codes if available
4. Contact support to disable 2FA

### Session Expired
**Symptoms**: Logged out unexpectedly

**Solutions**:
1. Log in again
2. Check "Remember Me" option
3. Disable browser privacy extensions temporarily
4. Check if cookies are enabled

## Time Tracking

### Timer Not Starting
**Symptoms**: Click start but nothing happens

**Solutions**:
1. Refresh the page
2. Check internet connection
3. Clear browser cache
4. Disable ad blockers
5. Try incognito/private mode
6. Use a different browser

### Time Entry Missing
**Symptoms**: Logged time doesn't appear

**Solutions**:
1. Check date filter (might be on different day)
2. Check project filter
3. Look in "All Projects" view
4. Check if entry was deleted
5. Verify sync completed

### Time Not Syncing
**Symptoms**: Entries on one device not showing on another

**Solutions**:
1. Refresh both devices
2. Check internet connection
3. Wait a few minutes
4. Log out and log back in

---

## Invoicing

### Invoice Not Sending
**Symptoms**: Send button doesn't work or client doesn't receive

**Solutions**:
1. Check client email address is correct
2. Ask client to check spam folder
3. Verify your email is verified
4. Try downloading and sending manually
5. Check email delivery logs in Settings

### Payment Link Not Working
**Symptoms**: Client can't pay through link

**Solutions**:
1. Verify payment processor is connected
2. Check if link has expired
3. Try generating a new invoice
4. Test payment link yourself
5. Contact support with invoice ID

### Invoice Amount Wrong
**Symptoms**: Calculations seem incorrect

**Solutions**:
1. Check tax settings
2. Verify line item quantities
3. Check discount application
4. Review currency settings
5. Recalculate manually to compare

---

## Projects

### Can't Create Project
**Symptoms**: Create button not working

**Solutions**:
1. Fill in all required fields
2. Check for validation errors (red text)
3. Verify you have permission
4. Refresh and try again
5. Check account limits

### Files Won't Upload
**Symptoms**: Upload stuck or fails

**Solutions**:
1. Check file size (max 100MB)
2. Check file type is supported
3. Try a different file
4. Check internet connection
5. Use a different browser

### Project Not Visible
**Symptoms**: Project disappeared

**Solutions**:
1. Check filter settings
2. Look in archived projects
3. Verify project access permissions
4. Check if project was deleted
5. Ask team admin about access

---

## Client Portal

### Client Can't Access Portal
**Symptoms**: Client says link doesn't work

**Solutions**:
1. Verify portal is enabled for client
2. Check client email is correct
3. Resend portal invitation
4. Check client's spam folder
5. Try generating new access link

### Client Can't See Project
**Symptoms**: Project not showing in portal

**Solutions**:
1. Enable client portal for that project
2. Check visibility settings
3. Verify client is assigned to project
4. Refresh the portal

---

## Payments

### Stripe Not Connected
**Symptoms**: Can't accept card payments

**Solutions**:
1. Complete Stripe onboarding
2. Verify Stripe account is activated
3. Check if Stripe is available in your country
4. Reconnect Stripe integration
5. Contact Stripe support

### Payment Not Received
**Symptoms**: Client paid but not showing

**Solutions**:
1. Wait 24-48 hours for processing
2. Check payment processor dashboard
3. Verify correct account connected
4. Check for payment holds
5. Contact support with details

### Wrong Amount Charged
**Symptoms**: Client charged incorrect amount

**Solutions**:
1. Check invoice details
2. Verify currency settings
3. Check for pending transactions
4. Issue refund if needed
5. Create credit note

---

## Integrations

### Integration Not Connecting
**Symptoms**: Can't complete OAuth/connection

**Solutions**:
1. Disable popup blockers
2. Allow third-party cookies
3. Try incognito mode
4. Use different browser
5. Check third-party service status

### Data Not Syncing
**Symptoms**: Connected but no data flowing

**Solutions**:
1. Check sync settings
2. Verify permissions granted
3. Trigger manual sync
4. Disconnect and reconnect
5. Check integration logs

### Calendar Events Missing
**Symptoms**: Events not appearing in KAZI

**Solutions**:
1. Check sync direction settings
2. Verify correct calendar selected
3. Check event visibility settings
4. Wait for sync interval
5. Trigger manual sync
