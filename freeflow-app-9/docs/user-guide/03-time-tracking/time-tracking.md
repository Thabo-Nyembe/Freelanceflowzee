# Time Tracking

Track every minute of billable work with KAZI's powerful time tracking features.

## Timer Basics

### Starting the Timer
**Method 1: Quick Start**
1. Click the **Timer** button in the top bar
2. Timer starts immediately
3. Add project/description later

**Method 2: From Time Tracking**
1. Go to **Time Tracking**
2. Enter description
3. Select project
4. Click **Start**

**Method 3: From Project**
1. Open a project
2. Click timer icon on any task
3. Timer starts for that task

**Method 4: Keyboard Shortcut**
- Press `Ctrl/Cmd + T` from anywhere

### Stopping the Timer
- Click the **Stop** button
- Time entry is automatically saved
- Edit details if needed

### Pausing
- Click **Pause** to take a break
- Timer state is preserved
- Click **Resume** to continue

## Manual Time Entry

### Adding Past Time
1. Go to **Time Tracking**
2. Click **+ Manual Entry**
3. Fill in:
   - Date
   - Start time
   - End time (or duration)
   - Project
   - Description
4. Click **Save**

### Bulk Entry
For logging multiple entries:
1. Click **Bulk Add**
2. Enter multiple rows
3. Submit all at once

## Time Entry Details

### Required Fields
- **Duration**: Hours and minutes
- **Date**: When work was done
- **Project**: Link to project

### Optional Fields
- **Task**: Specific task within project
- **Description**: What was done
- **Billable**: Whether to invoice
- **Tags**: Categorization
- **Notes**: Additional details

## Billable vs Non-Billable

### Billable Time
- Counts toward invoices
- Tracked in project budget
- Shown to clients (if enabled)

### Non-Billable Time
- Internal work
- Administrative tasks
- Learning/training
- Non-chargeable meetings

### Setting Default
1. Go to **Settings > Time Tracking**
2. Set **Default Billable Status**
3. Override per entry as needed

## Automatic Tracking

### Idle Detection
1. Go to **Settings > Time Tracking**
2. Enable **Idle Detection**
3. Set idle threshold (e.g., 5 minutes)
4. Choose action:
   - Pause timer
   - Discard idle time
   - Keep tracking

## Troubleshooting

### Timer Not Starting
- Check internet connection
- Refresh the page
- Clear browser cache
- Try incognito mode

### Time Entry Missing
- Check date filter
- Verify project selection
- Look in "All Projects" view
- Check if deleted accidentally

### Sync Issues
- Refresh the page
- Check browser extension
- Verify calendar connection
- Contact support if persists
