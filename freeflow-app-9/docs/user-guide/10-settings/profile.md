# Profile Settings

Configure your KAZI profile for the best experience.

## Personal Information

### Basic Details
- **Full Name**: Displayed to clients and team
- **Profile Photo**: Professional headshot recommended
- **Job Title**: Appears on invoices and proposals
- **Bio**: Brief professional description

### Contact Information
- Email address
- Phone number
- Location
- Timezone (for scheduling)

## Business Information

### Company Details
- **Business Name**: Used on invoices
- **Business Address**: Official documents
- **Tax ID / VAT**: Required for tax-compliant invoicing

### Professional Details
- Industry/specialization
- Years of experience
- Portfolio URL
- Social media links

## Billing Defaults

### Rates
- Default hourly rate
- Preferred currency
- Rate visibility settings

### Payment Terms
- Default payment terms (Net 15, 30, etc.)
- Late fee settings
- Accepted payment methods

## Preferences

### Language
- Interface language
- Date format
- Number format

### Notifications
- Email notifications
- Push notifications
- Notification frequency

### Privacy
- Profile visibility
- Activity status
- Data sharing preferences

## Updating Your Profile

1. Click avatar in top right
2. Select **Profile Settings**
3. Make changes
4. Click **Save**

## Tips

- Use a professional photo
- Keep contact info current
- Set accurate timezone
- Review billing defaults regularly
