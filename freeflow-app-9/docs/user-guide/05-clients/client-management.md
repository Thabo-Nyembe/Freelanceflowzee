# Client Management

Manage your clients effectively with KAZI's client management features.

## Adding Clients

### New Client
1. Go to **Clients** in the sidebar
2. Click **+ Add Client**
3. Fill in details:
   - Company name
   - Contact person
   - Email address
   - Phone number
   - Address
   - Currency preference

### Import Clients
- Import from CSV
- Sync from QuickBooks/Xero
- Migrate from other platforms

## Client Portal

### Enabling Portal Access
1. Open client profile
2. Click **Settings**
3. Enable **Client Portal**
4. Send invitation

### What Clients See
Configure visibility for:
- Project progress
- Task status
- Time entries
- Files and documents
- Invoices
- Messages

### Client Actions
Clients can:
- View project updates
- Approve milestones
- Pay invoices online
- Upload files
- Send messages

## Client Organization

### Tags & Categories
- Add tags for easy filtering
- Create client categories
- Track client type (Retainer, Project-based)

### Notes & History
- Add internal notes
- View activity history
- Track communications

## Client Analytics

### Revenue Tracking
- Total billed
- Outstanding balance
- Average project value
- Payment history

### Project History
- All projects with client
- Completion rates
- Total hours worked

## Best Practices

1. Keep contact info updated
2. Set clear expectations early
3. Use the portal for transparency
4. Track all communications
5. Regular check-ins for long-term clients
