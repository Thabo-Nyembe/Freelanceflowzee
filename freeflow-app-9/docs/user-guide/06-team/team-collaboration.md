# Team Collaboration

Work effectively with team members in KAZI.

## Inviting Team Members

### Send Invitation
1. Go to **Team** in sidebar
2. Click **+ Invite Member**
3. Enter email address
4. Select role
5. Click **Send Invite**

### Roles
- **Admin**: Full access to everything
- **Manager**: Manage projects and team
- **Member**: Work on assigned projects
- **Viewer**: Read-only access

## Role Permissions

| Permission | Admin | Manager | Member | Viewer |
|------------|-------|---------|--------|--------|
| Manage team | ✓ | - | - | - |
| Billing access | ✓ | - | - | - |
| Create projects | ✓ | ✓ | - | - |
| Assign tasks | ✓ | ✓ | ✓ | - |
| Track time | ✓ | ✓ | ✓ | - |
| View reports | ✓ | ✓ | ✓ | ✓ |

## Project Assignment

### Assign to Projects
1. Open project settings
2. Click **Team Members**
3. Add members
4. Set project-specific permissions

### Task Assignment
- Assign tasks to team members
- Set due dates and priorities
- Track individual progress
