# Project Management

Master KAZI's project management features to organize work, track progress, and deliver projects on time.

## Creating Projects

### New Project Wizard
1. Navigate to **Projects Hub**
2. Click **+ New Project**
3. Complete the setup:

#### Basic Information
- **Project Name**: Clear, descriptive title
- **Project Code**: Auto-generated or custom (e.g., PRJ-001)
- **Client**: Select from your clients
- **Description**: Project scope and objectives

#### Timeline
- **Start Date**: When work begins
- **End Date**: Target completion date
- **Milestones**: Key deliverables with dates

#### Budget & Billing
- **Billing Type**:
  - Hourly (track time, bill per hour)
  - Fixed Price (set total amount)
  - Retainer (recurring monthly)
- **Budget**: Maximum spend limit
- **Hourly Rate**: Override default rate if needed

#### Team
- **Project Owner**: Primary responsible person
- **Team Members**: Assign collaborators
- **Access Level**: Viewer, Editor, Admin

## Project Views

### Kanban Board
Visual workflow with customizable columns:
- **To Do**: Tasks not started
- **In Progress**: Active work
- **Review**: Awaiting approval
- **Done**: Completed tasks

Drag and drop tasks between columns.

### List View
Traditional task list with:
- Sorting by due date, priority, assignee
- Filtering by status, tags, dates
- Bulk actions for multiple tasks

### Calendar View
Timeline-based view showing:
- Task due dates
- Milestones
- Team availability
- Deadlines

### Gantt Chart
Project timeline visualization:
- Task dependencies
- Critical path highlighting
- Resource allocation
- Progress tracking

## Task Management

### Creating Tasks
1. Click **+ Add Task** in any view
2. Enter task details:
   - **Title**: What needs to be done
   - **Description**: Details and requirements
   - **Assignee**: Who's responsible
   - **Due Date**: When it's due
   - **Priority**: Low, Medium, High, Urgent
   - **Tags**: Categorization labels

### Task Features
- **Subtasks**: Break down complex tasks
- **Checklists**: Step-by-step items
- **Attachments**: Files and documents
- **Comments**: Team discussion
- **Time Tracking**: Log time directly
- **Dependencies**: Link related tasks

### Task Automation
Set up automatic actions:
- Move to "In Progress" when timer starts
- Notify assignee when task is assigned
- Move to "Done" when time is logged
- Send reminder before due date

## Milestones

### Setting Milestones
1. Go to project **Timeline** tab
2. Click **+ Add Milestone**
3. Define:
   - Milestone name
   - Target date
   - Description
   - Linked tasks
   - Payment trigger (optional)

### Milestone Billing
Link payments to milestones:
- Automatic invoice when milestone is complete
- Partial payment releases
- Client approval workflow

## Project Templates

### Creating Templates
1. Go to **Settings > Project Templates**
2. Click **+ New Template**
3. Define:
   - Template name
   - Default tasks and milestones
   - Standard workflow stages
   - Team roles

### Using Templates
1. Click **+ New Project**
2. Select **From Template**
3. Choose template
4. Customize as needed

### Pre-built Templates
- **Web Development**: Design, Development, Testing, Launch
- **Content Creation**: Research, Draft, Review, Publish
- **Marketing Campaign**: Strategy, Creation, Launch, Analysis
- **Consulting**: Discovery, Analysis, Recommendations, Implementation

## Client Collaboration

### Client Portal Access
Give clients visibility into their projects:
1. Go to project **Settings**
2. Enable **Client Portal**
3. Choose what clients can see:
   - Project progress
   - Task status
   - Time entries
   - Files
   - Messages

### Client Feedback
- Comment on deliverables
- Approve milestones
- Request revisions
- Upload reference files

## Project Analytics

### Progress Tracking
- Completion percentage
- Tasks completed vs remaining
- Time spent vs budgeted
- Budget consumption

### Reports
- **Project Summary**: Overview of status
- **Time Report**: Hours by task/person
- **Budget Report**: Spend vs budget
- **Team Performance**: Individual contributions

## Best Practices

### Project Setup
1. Define clear scope before starting
2. Break work into manageable tasks
3. Set realistic deadlines
4. Assign clear ownership

### During Project
1. Update task status regularly
2. Log time as you work
3. Communicate with clients proactively
4. Address blockers quickly

### Project Completion
1. Review all tasks are done
2. Get client sign-off
3. Generate final invoice
4. Archive project
5. Conduct retrospective
