# Quick Start Guide

Get up and running with KAZI in under 10 minutes. This guide covers the essential steps to start managing your freelance business.

## 5-Minute Setup

### Step 1: Create a Client (1 min)
1. Click **Clients** in the sidebar
2. Click **+ Add Client**
3. Enter client details:
   - Name and company
   - Email address
   - Phone (optional)
4. Click **Save**

### Step 2: Create a Project (2 min)
1. Click **Projects Hub** in the sidebar
2. Click **+ New Project**
3. Fill in project details:
   - Project name
   - Select client
   - Set deadline
   - Choose billing type (Hourly/Fixed)
4. Click **Create Project**

### Step 3: Start Tracking Time (1 min)
1. Click **Time Tracking** in the sidebar
2. Click the **Start Timer** button
3. Select your project
4. Add a description (optional)
5. Timer starts automatically!

### Step 4: Create an Invoice (1 min)
1. Click **Invoices** in the sidebar
2. Click **+ New Invoice**
3. Select client and project
4. Time entries are auto-populated
5. Click **Send Invoice**
