# Account Setup Guide

This guide walks you through setting up your KAZI account for the first time.

## Creating Your Account

### Step 1: Sign Up
1. Visit [kazi.com](https://kazi.com)
2. Click "Get Started Free" or "Sign Up"
3. Choose your signup method:
   - Email and password
   - Google account
   - GitHub account

### Step 2: Verify Your Email
1. Check your inbox for the verification email
2. Click the verification link
3. You'll be redirected to complete your profile

### Step 3: Complete Your Profile

#### Personal Information
- **Full Name**: Your display name
- **Profile Photo**: Upload a professional photo
- **Bio**: Brief description of your services
- **Location**: Your city/country
- **Timezone**: For accurate scheduling

#### Professional Details
- **Job Title**: e.g., "Freelance Web Developer"
- **Skills**: Add your key skills
- **Hourly Rate**: Your default billing rate
- **Currency**: Your preferred currency

#### Business Information (Optional)
- **Business Name**: For invoices and contracts
- **Business Address**: For official documents
- **Tax ID**: For invoicing compliance

## Setting Up Payment Methods

### Receiving Payments
1. Go to **Settings > Payment Methods**
2. Connect your payment accounts:
   - **Stripe**: For credit/debit card payments
   - **PayPal**: For PayPal payments
   - **Bank Transfer**: Add bank details
   - **Crypto**: Connect crypto wallet

### Payment Settings
- Default payment terms (Net 15, Net 30, etc.)
- Late payment fees
- Accepted payment methods
- Currency preferences

## Customizing Your Workspace

### Dashboard Layout
1. Go to **Settings > Appearance**
2. Choose your theme (Light/Dark/System)
3. Customize sidebar items
4. Set your default dashboard view

### Notification Preferences
1. Go to **Settings > Notifications**
2. Configure email notifications:
   - New messages
   - Payment received
   - Project updates
   - Deadline reminders
3. Set up push notifications
4. Choose notification frequency

### Email Templates
1. Go to **Settings > Email Templates**
2. Customize templates for:
   - Invoice emails
   - Payment reminders
   - Project updates
   - Welcome messages

## Security Setup

### Two-Factor Authentication (2FA)
1. Go to **Settings > Security**
2. Click "Enable 2FA"
3. Scan QR code with authenticator app
4. Enter verification code
5. Save backup codes

### Password Requirements
- Minimum 12 characters
- At least one uppercase letter
- At least one number
- At least one special character

### Active Sessions
- View all logged-in devices
- Revoke access to unknown devices
- Set session timeout preferences

## Importing Data

### From Other Platforms
KAZI supports importing from:
- **Toggl**: Time tracking data
- **FreshBooks**: Invoices and clients
- **Harvest**: Projects and time entries
- **CSV Files**: Custom data import

### How to Import
1. Go to **Settings > Import Data**
2. Select your source platform
3. Authorize the connection or upload file
4. Map fields to KAZI fields
5. Review and confirm import

## Troubleshooting

### Common Setup Issues

**Email not received?**
- Check spam/junk folder
- Add noreply@kazi.com to contacts
- Request resend from login page

**Payment connection failed?**
- Ensure popup blocker is disabled
- Try a different browser
- Contact support with error details

**Import failed?**
- Check file format requirements
- Ensure all required fields are present
- Try importing smaller batches
