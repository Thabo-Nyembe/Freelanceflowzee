# KAZI Demo Video Recording Guide

## Technical Setup

### Recording Software Options
- **macOS**: QuickTime (free), ScreenFlow ($149), OBS (free)
- **Windows**: OBS (free), Camtasia ($250), Loom (free tier)
- **Cross-platform**: Loom, CloudApp, Screencast-O-Matic

### Recommended Settings
- **Resolution**: 1920x1080 (1080p) minimum, 2560x1440 (1440p) preferred
- **Frame Rate**: 30fps (60fps for smooth UI animations)
- **Audio**: External microphone recommended (Blue Yeti, Rode NT-USB)
- **Browser**: Chrome (latest) in incognito mode (clean UI)

### Pre-Recording Checklist
- [ ] Close unnecessary applications
- [ ] Disable notifications (Do Not Disturb)
- [ ] Clear browser cache
- [ ] Use demo mode URL: `http://localhost:3000/dashboard?demo=true`
- [ ] Set browser zoom to 100%
- [ ] Hide bookmarks bar
- [ ] Test microphone levels

## Recording Tips

### Pacing
- Pause briefly after each action (1-2 seconds)
- Let animations complete before moving on
- Don't rush through forms

### Mouse Movement
- Move cursor smoothly, not erratically
- Hover over elements before clicking
- Use cursor highlighter if available

### Narration
- Record audio separately for better quality
- Speak clearly and at moderate pace
- Leave room for editing

### Common Mistakes to Avoid
- Scrolling too fast
- Clicking before cursor is in frame
- Forgetting to show success messages
- Background notifications appearing
